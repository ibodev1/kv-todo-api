## Todo API
Todo App API with Deno KV & Hono

## Development
If you're using [drux](https://deno.land/x/drux)
```bash
  drux dev
```
or
```bash
  deno task dev
```

## Test And Bench
### Test
```bash
  drux test
```
### Bench
```bash
  drux bench
```

You need [https://deno.land/x/drux](https://deno.land/x/drux)
